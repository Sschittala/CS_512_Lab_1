# Lab 1 - Project Overview

This repository contains the code, data, and results for Lab 1.

Authors: Tyler Strach, Miles Woodcock-Girard, Sai Chittala

## Folder Structure

### `code/`
Contains all Python scripts and helper modules for the project:
- **See README.md located in `code/` for more detail**
- **q1_… to q5_…** — Scripts implementing individual questions and experiments.
- **svm_hmm/** — Contains the SVM-HMM implementation, including source files, Makefiles, and classifiers.


### `data/`
Stores input datasets and transformed data files used for training and testing models:
- Original dataset (`train.txt`, `test.txt`)
- Transformed data for different models (`train_struct.txt`, `train_mc.txt`, etc.)
- Dataset descriptions (`fields_crf.txt`, `fields_struct.txt`)

### `latex/`
LaTeX files for documenting the assignment and generating report:
- **Lab_1.tex** — Main report file.
- **Figures/** — Figures, charts, and images referenced in the report. Including generated plots for various questions
- Supporting LaTeX styles and auxiliary files.

### `result/`
Stores outputs generated by running the code:
- Q1c `q1_decode.py` - (`decode_output.txt`)
- Q2a `q2_gradient.py` - (`gradient.txt`)
- Q2b `q2_gradient.py` - (`solution.txt`, `prediction.txt`)


## Usage
1. Place or generate necessary input files in `data/`.
2. Run scripts from `code/` to train models and produce results.
3. Generated figures and output files will be stored in `result/`.
4. Compile `latex/Lab_1.tex` to generate the final PDF report.
